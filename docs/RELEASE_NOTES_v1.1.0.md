# Acrylic Calendar 1.1

Version 1.1 contains one user-facing change: the Settings window visual redesign.

## Changed

- Reworked the Settings window from a macOS-like preferences panel into a Windows 10 Settings-inspired page.
- Preserved the existing settings and behavior unchanged.
- Added Windows 10-style dark UWP layout, rectangular controls, and custom toggle visuals.
- Kept the standard macOS window frame for correct system behavior.
- Made the Settings layout responsive to window width: wide windows use two columns, narrower windows resize controls and can collapse to one column.

## Release Files

- `Acrylic-calendar-macOS-universal.zip`: universal macOS app bundle.
- `Acrylic-calendar-binaries.zip`: standalone binaries.
- `SHA256SUMS.txt`: checksums.
